# Realistic Traffic Bot

Advanced human-like website traffic simulator for testing purposes.

## Features
- Real browser automation (Selenium)
- Human-like scrolling and clicking
- Proxy rotation
- Realistic browsing patterns
- Detailed logging

## Setup

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Edit `config.py` with your website URL
4. Run: `python main.py`

## Important
Only use this on websites YOU OWN for testing purposes!
